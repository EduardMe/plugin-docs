# NotePlan Plugins Documentation

NotePlan has a powerful plugin architecture that allows for the creation of many useful tools and additional functionality. Here you will find documentation for all of the currently available plugins for NotePlan.

On the left, you will see a list of pages available. We recommend you start with the <Link href="pluginlist">Plugin List</Link>, or use the search function at the top of the page to find a specific plugin.

Below the Plugin List, you will find the documentation for each individual plugin. And below that find the <a href="http://localhost:3000/templating" target="_blank">Templating</a> documentation.

## How to install plugins?

**On Mac:** Open the preferences of NotePlan and head over to "AI & Plugins". Then let NotePlan load the available plugins.

**On iOS:** Tap on the magnifiying glass icon bottom right and type "/install plugin".

## How to use plugins?

Once you have the plugin installed, open the command bar and type "/" (slash). All the available commands of every installed plugin will be loaded. Keep typing to search for either a specific plugin or a specific command.
