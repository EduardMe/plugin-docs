# Modules Overview
`Templating` includes a suite of modules which are available for inclusion in any template. Each module has a specific category (aka "namespace") which is used within your templates.

## Modules
The following modules are included in `Templating`

### Date Module
The <Link href="templating-modules-date" alt="Date Module"> Date Module </Link> provides methods and variables which work with dates.

### Time Module
The <Link href="templating-modules-time" alt="Time Module">Time Module </Link> provides methods and variables which work with dates.

### Note Module
The <Link href="templating-modules-note" alt="Note Module">Note Module</Link> provides a suite of methods which provide access to current note attributes

### System Module
The <Link href="templating-modules-system" alt="System Module">System Module</Link> provides a suite of methods which provide access to system level commands such as clipboard or current selection

### Utility Module
The <Link href="templating-modules-utility" alt="Utility Module">Utility Module</Link> provides a suite of utility methods which can be used in your templates

### Web Module
The <Link href="templating-modules-web" alt="Web Module">Web Module</Link> provides a suite of methods which provide access to web APIs such as advice, affirmation or weather.

## Examples
The following are some examples of how you can use these modules within your templates.

```markdown
// inserts current date
<%- date.now() %>
```

```markdown
// inserts current time
<%- time.now() %>
```

```markdown
// insert contents of clipboard
<%- system.clipboard() %>
````

```markdown
// format string with titleCase
<%- utils.titleCase('mike') %>
```

```markdown
// return random advice
<%- web.advice('mike') %>
```
