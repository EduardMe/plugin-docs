# Modules
While you can surely use templates which contain simple content, the real power of using a templating system is the ability to use custom variables and methods within the template content.

## Built-in Modules (variables and methods)
`np.Templating` includes a suite of built-in modules which are organized to provide groups of variables and methods.

The following is a quick overview of all the included modules, for more details visit the <Link href="templating-modules-overview">Templating Modules - Overview</Link> section.

### Date Module
<Link href="templating-modules-date">Templating Modules - Date</Link> contains every internal variable / method related to dates.

### Frontmatter Module
<Link href="templating-modules-frontmatter">Templating Modules - Frontmatter</Link> exposes all the frontmatter variables of a file as internal variables.

### Note Module
<Link href="templating-modules-note">Templating Modules - Note</Link> exposes all the note variables of a file as internal variables.

### Time Module
<Link href="templating-modules-time">Templating Modules - Time</Link> contains every internal variable / method related to time values.

### Web Module
<Link href="templating-modules-web">Templating Modules - Web</Link> contains every internal variable / method related to the web (using web requests).
